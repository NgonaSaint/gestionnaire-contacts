# Gestionnaire de Contacts

Projet React réalisé dans le cadre du module de développement Front-End à l'**ESTIAM Paris**.

## Objectif

Créer une application React permettant de gérer une liste de contacts :
- Ajouter un contact
- Afficher la liste des contacts
- Modifier un contact
- Supprimer un contact

---

## Bonus réalisés

- Recherche en temps réel par prénom, nom ou email
- Tri des contacts par nom (A→Z / Z→A)
- Validation des champs (email valide, champs obligatoires)
